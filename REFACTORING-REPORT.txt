8. 핵심 비즈니스 로직 요약
================================================================================

[시뮬레이터 계산 파이프라인]
  입력 (FullForm)
    |
    v
  홀 매출 = 좌석 * 객단가 * 회전율 * (평일일수 + 주말일수*주말배율) * 4.345주
  배달 매출 = 배달매출액 * (1 - 배달수수료율)
  총 매출 = 홀 매출 + 배달 매출
    |
    v
  총 비용:
    원가        = 총매출 * 원가율
    인건비      = 직접입력 또는 (직원수*월급+알바시급*시간*일수)
    임대료      = 월세 + 관리비
    공과금      = 전기/수도/가스
    마케팅      = 월 마케팅비
    기타비용    = 소모품 등
    카드수수료  = 총매출 * 수수료율
    로열티      = 프랜차이즈 시 (총매출 * 로열티율)
    |
    v
  세전 순이익 = 총매출 - 총비용
  세후 순이익 = 세전 순이익 * (1 - 세율)
  현금흐름    = 세후 순이익 - 월 대출상환액
  손익분기점  = 총비용 / (1 - 원가율)
  투자금 회수 = 총초기비용 / 월 세후순이익

[게임 일일 운영 계산]
  배율 계산:
    날씨 배율   = WX[weather].mod (0.65 ~ 1.1)
    평판 배율   = 0.6 + (rep/100) * 0.8
    스킬 배율   = 0.65 + (avgSkill/100) * 0.7
    주말 배율   = (day%7 >= 5) ? 1.4 : 1.0
    효과 배율   = efx[] 합산

  고객수 = base * 모든배율 * (0.8 ~ 1.2 랜덤)
  매출   = 고객수 * 객단가 * 객단가배율
  비용   = 원가 + 인건비 + 고정비(30일마다)
