# VELA 경영 분석 엔진 — B2B 파트너십 제안서

---

## 1. 제안 요약

POS 매출 데이터를 실시간 경영 인사이트로 전환하는 분석 엔진을 귀사 POS 시스템에 통합 제안합니다.

**핵심 가치:** 귀사 사장님 고객에게 "매출 집계"를 넘어 "경영 분석"을 제공하여 차별화 + 이탈 방지

---

## 2. 시장 현황

| 구분 | 현재 POS | VELA 통합 후 |
|------|---------|-------------|
| 매출 조회 | 일별/메뉴별 집계 | 동일 |
| 원가 분석 | X | 메뉴별 원가율, 건당 순익 자동 계산 |
| 손익 분석 | X | 인건비/임대료/세금 포함 실제 순이익 |
| 손익분기점 | X | BEP 자동 계산 + 달성까지 D-day |
| AI 전략 | X | 객단가/회전율/원가 최적화 추천 |
| 세금 예측 | X | 부가세/종소세 예상액 |
| 경쟁 비교 | X | 업종 평균 대비 내 매장 포지션 |

**결론:** 모든 POS가 "얼마 팔렸냐"는 보여주지만, "얼마 남았냐"는 아무도 안 보여줍니다.

---

## 3. 통합 방식

### Option A: API 연동 (권장)
- 귀사 POS에서 VELA API로 매출 데이터 전송
- VELA가 분석 결과 JSON 반환
- 귀사 앱/대시보드에서 결과 표시

```
[귀사 POS] → 매출 데이터 → [VELA API] → 분석 결과 → [귀사 대시보드]
```

### Option B: 화이트라벨 대시보드
- VELA 대시보드를 귀사 브랜드로 커스터마이징
- iframe 또는 서브도메인으로 제공
- 귀사 로고/색상 적용

### Option C: SDK 임베딩
- React 컴포넌트 SDK 제공
- 귀사 앱에 직접 임베딩
- 완전한 UI 커스터마이징 가능

---

## 4. 분석 엔진 기능

### 실시간 분석 (매출 데이터 입력 시 즉시)
- 일별/주별/월별 매출 추이
- 메뉴별 원가율 및 수익성 순위
- 시간대별 매출 패턴
- 요일별 매출 패턴

### 경영 인사이트 (월 1회 자동 생성)
- 손익계산서 자동 생성
- 손익분기점 계산 및 달성률
- 업종 평균 대비 벤치마크
- 인건비/임대료 비율 진단

### AI 전략 추천
- 원가 절감 포인트 도출
- 객단가 최적화 전략
- 메뉴 구성 최적화
- 시즌별 매출 예측

## 6. 기대 효과 (귀사)

1. **차별화:** 경쟁 POS 대비 "경영 분석" 기능 독점 제공
